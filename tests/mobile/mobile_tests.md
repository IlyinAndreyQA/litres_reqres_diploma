# Mobile автотесты Litres

## Проверяемые сценарии

1. Поиск книги в мобильном приложении.
2. Открытие каталога.
3. Открытие экрана профиля/авторизации.

## Запуск

Перед запуском должен быть доступен Appium Server и Android-эмулятор или BrowserStack/Selenoid mobile окружение.

```bash
pytest tests/mobile
```
