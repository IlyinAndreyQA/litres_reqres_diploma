# UI автотесты Litres

## Проверяемые сценарии

1. Пользователь находит книгу по названию.
2. Пользователь находит книги по автору.
3. Пользователь переходит в раздел аудиокниг.
4. Пользователь переходит в каталог жанров.
5. Пользователь открывает раздел популярных книг.
6. Пользователь открывает корзину.
7. Пользователь открывает раздел абонементов.
8. Форма авторизации показывает валидацию для некорректного email.

## Запуск

```bash
pytest tests/web
```
