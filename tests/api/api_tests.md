# API автотесты Reqres

## Проверяемые сценарии

1. Получение списка пользователей.
2. Получение пользователя по id.
3. Создание пользователя.
4. Обновление пользователя.
5. Удаление пользователя.
6. Ошибка регистрации без пароля.

## Модели

Для формирования request body и десериализации response body используются Pydantic-модели из `litres_reqres_diploma/models/reqres.py`.

## Запуск

```bash
pytest tests/api
```
