# Ручные тест-кейсы для Allure TestOps

Эти кейсы можно завести вручную в Allure TestOps и связать с автотестами после первого импорта результатов.

## WEB

1. Поиск книги по названию
   - Открыть главную страницу Litres.
   - Ввести название книги в поиск.
   - Проверить, что в результатах есть найденная книга.

2. Открытие корзины
   - Открыть главную страницу Litres.
   - Перейти в корзину.
   - Проверить, что отображается страница корзины.

3. Валидация формы авторизации
   - Открыть главную страницу Litres.
   - Открыть форму входа.
   - Ввести некорректную почту.
   - Проверить сообщение об ошибке.

## API

1. Получение списка пользователей
   - Отправить GET `/users?page=2`.
   - Проверить статус `200`.
   - Проверить структуру и данные ответа.

2. Создание пользователя
   - Отправить POST `/users` с телом запроса.
   - Проверить статус `201`.
   - Проверить данные созданного пользователя.

## MOBILE

1. Поиск книги в приложении
   - Открыть мобильное приложение Litres.
   - Перейти в поиск.
   - Ввести название книги.
   - Проверить результаты поиска.
